# Student Attendance System

A simple and interactive Student Attendance System developed using HTML, CSS, and JavaScript.

## Project Overview

The Student Attendance System is a front-end web application designed to simplify attendance management in an educational environment. The system allows users to:

- Add classes
- Add students
- Mark attendance
- View attendance summary instantly

This project is developed for academic learning purposes and demonstrates the use of basic web development technologies.

---

## Features

- Add new classes dynamically
- Add student details
- Mark students as:
  - Present
  - Absent
  - Leave
- Real-time attendance summary
- Simple and user-friendly interface
- Responsive design using CSS
- Local storage support using JavaScript

## How It Works

1. Open the project in a web browser.
2. Add a class using the **Add Class** button.
3. Add students to the selected class.
4. Mark attendance using:
   - A → Absent
   - P → Present
   - L → Leave
5. View attendance summary instantly.
6. Submit attendance for the selected class.
